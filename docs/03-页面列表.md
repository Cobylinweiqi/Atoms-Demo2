# 03. 页面列表

> **Nova Studio — AI Full Stack Builder**  
> 版本: v1.0 | 日期: 2026-07-16

---

## 3.1 营销页面（公开）

| 路由 | 页面 | 说明 |
|---|---|---|
| `/` | 首页 | Hero + 功能展示 + 示例 + CTA |
| `/pricing` | 定价页 | 各层级对比表 |
| `/templates` | 模板库 | 可用项目模板画廊 |
| `/showcase` | 案例展示 | 用户作品展示 |
| `/docs` | 文档 | 使用文档 + API 文档 |
| `/changelog` | 更新日志 | 版本更新记录 |
| `/about` | 关于 | 团队介绍 |
| `/blog` | 博客 | 技术博客（可选） |

## 3.2 认证页面

| 路由 | 页面 | 说明 |
|---|---|---|
| `/login` | 登录 | 邮箱密码 + OAuth（GitHub/Google） |
| `/register` | 注册 | 注册表单 + OAuth |
| `/forgot-password` | 忘记密码 | 发送重置邮件 |
| `/reset-password` | 重置密码 | 输入新密码 |
| `/oauth/github/callback` | GitHub OAuth 回调 | 处理 GitHub OAuth |
| `/oauth/google/callback` | Google OAuth 回调 | 处理 Google OAuth |
| `/invite/:token` | 邀请接受 | 接受团队邀请 |

## 3.3 应用主页面（需认证）

| 路由 | 页面 | 说明 |
|---|---|---|
| `/app` | 控制台首页 | 重定向到默认组织 |
| `/app/[orgSlug]` | 组织控制台 | 项目列表 + 最近活动 + 快速开始 |
| `/app/[orgSlug]/projects` | 项目列表 | 卡片/列表视图切换、搜索、筛选 |
| `/app/[orgSlug]/projects/new` | 新建项目 | 模板选择 + AI 描述 + 配置 |
| `/app/[orgSlug]/settings` | 组织设置 | 基本信息 + 偏好 |
| `/app/[orgSlug]/members` | 成员管理 | 成员列表 + 邀请 + 角色管理 |
| `/app/[orgSlug]/billing` | 账单管理 | 订阅 + 用量 + 发票 |
| `/app/[orgSlug]/activity` | 活动日志 | 操作审计记录 |

---

## 3.4 项目工作区页面

| 路由 | 页面 | 说明 |
|---|---|---|
| `/app/[orgSlug]/p/[projectSlug]` | 工作区主界面 | 多标签页：Chat/Code/Visual/Preview/Theme/Deploy |
| `/app/[orgSlug]/p/[projectSlug]/preview` | 全屏预览 | 独立预览窗口 |
| `/app/[orgSlug]/p/[projectSlug]/agents` | Agent 构建 | 流程编排画布 |
| `/app/[orgSlug]/p/[projectSlug]/settings` | 项目设置 | 框架/环境变量/域名 |
| `/app/[orgSlug]/p/[projectSlug]/github` | GitHub 同步 | 仓库关联 + 同步 |
| `/app/[orgSlug]/p/[projectSlug]/deployments` | 部署历史 | 部署记录 + 日志 |

### 工作区标签页说明

| 标签 | 功能 |
|---|---|
| Chat | AI 对话面板，流式交互，工具调用展示 |
| Code | Monaco 代码编辑器，文件树，搜索替换 |
| Visual | 可视化编辑器，组件树 + 画布 + 属性面板 |
| Preview | 实时预览 iframe，设备框架，控制台日志 |
| Theme | 主题编辑器，颜色/排版/间距/圆角/阴影 |
| Components | 组件库浏览，自定义组件管理 |
| Agents | Agent 流程编排（ReactFlow） |
| Deploy | 部署面板，平台选择，环境变量，构建日志 |
| GitHub | 仓库关联，推送/拉取，分支管理 |
| Settings | 项目设置，框架，环境变量，域名 |

## 3.6 页面路由架构（Next.js App Router）

```
src/app/
├── (marketing)/                    # 营销页面组（公开）
│   ├── page.tsx                    # 首页
│   ├── pricing/page.tsx
│   ├── templates/page.tsx
│   ├── showcase/page.tsx
│   └── docs/page.tsx
│
├── (auth)/                         # 认证页面组
│   ├── login/page.tsx
│   ├── register/page.tsx
│   └── forgot-password/page.tsx
│
├── (app)/                          # 认证后应用
│   ├── layout.tsx                  # 应用 Shell（侧边栏、顶栏）
│   ├── page.tsx                    # 控制台首页
│   │
│   ├── [orgSlug]/
│   │   ├── layout.tsx             # 组织 Shell
│   │   ├── page.tsx                # 组织控制台
│   │   ├── projects/
│   │   │   ├── page.tsx            # 项目列表
│   │   │   └── new/page.tsx        # 新建项目向导
│   │   ├── settings/page.tsx
│   │   ├── members/page.tsx
│   │   ├── billing/page.tsx
│   │   └── activity/page.tsx
│   │
│   ├── [orgSlug]/p/[projectSlug]/
│   │   ├── layout.tsx              # 工作区 Shell
│   │   ├── page.tsx                # Chat 标签（默认）
│   │   ├── code/page.tsx           # 代码编辑器
│   │   ├── visual/page.tsx        # 可视化编辑器
│   │   ├── preview/page.tsx       # 实时预览
│   │   ├── theme/page.tsx          # 主题编辑器
│   │   ├── components/page.tsx     # 组件库
│   │   ├── agents/page.tsx        # Agent 编排
│   │   ├── deploy/page.tsx        # 部署面板
│   │   ├── github/page.tsx        # GitHub 同步
│   │   ├── deployments/page.tsx   # 部署历史
│   │   └── settings/page.tsx      # 项目设置
│   │
│   └── settings/                   # 用户设置
│       ├── page.tsx
│       ├── api-keys/page.tsx
│       ├── preferences/page.tsx
│       ├── notifications/page.tsx
│       └── appearance/page.tsx
│
├── api/                            # BFF API 路由
│   ├── auth/[...nextauth]/route.ts
│   ├── ai/stream/route.ts          # SSE 流式代理
│   └── webhooks/
│       └── stripe/route.ts
│
└── layout.tsx                      # 根布局
```
