## 1. 设计哲学

```
不只是暗色主题 — 是有深度的暗色

不只是毛玻璃 — 是有质感的玻璃

不只是渐变 — 是会流动的光

不只是圆角 — 是柔软的触感

不只是阴影 — 是漂浮的存在感
```

### 核心原则

| 原则 | 说明 |
|---|---|
| **Depth through layers** | 不是扁平黑底，而是多层半透明表面叠加，营造空间感 |
| **Light that breathes** | 渐变不是静态色块，是缓慢流动的光晕 |
| **Glass, not plastic** | 玻璃有内边缘高光（inset highlight），不是廉价塑料感 |
| **Whisper, don't shout** | 渐变点缀而非铺满，克制使用品牌色 |
| **Generous space** | 大间距、大圆角、大字号 — 空间即奢华 |
| **Texture matters** | 极微噪点叠加，消除渐变条带，增加材质感 |
| **Motion with physics** | 动画遵循物理曲线，有惯性、有减速、有弹性 |

### 与竞品的差异化

| 维度 | Apple | Linear | Vercel | **Nova Studio** |
|---|---|---|---|---|
| 背景 | 纯色 + 模糊层 | 极暗纯色 | 纯黑 + 网格 | **暗色 + 流动光晕 + 点阵** |
| 玻璃 | 毛玻璃原生 | 克制玻璃 | 无 | **有质感的浮空玻璃** |
| 渐变 | 多彩 | 紫色克制 | 黑白对比 | **紫→蓝→青 流动渐变** |
| 圆角 | 12-22px | 6-12px | 6-8px | **16-32px 大圆角** |
| 字体 | SF Pro | Inter | Geist | **Satoshi + Inter + JetBrains** |
| 噪点 | 无 | 有（微） | 无 | **有（极微）** |
| 阴影 | 柔和 | 深色硬 | 几乎无 | **多层柔阴影 + 内高光** |

## 2. 颜色系统

### 2.1 背景层

不是单一黑色，而是有蓝紫色温的深色体系，模拟"深邃太空"的质感。

```
┌─────────────────────────────────────────────────────┐
│  Layer 0 — Base          #07070B  最深层（页面底色）  │
│  Layer 1 — Surface 1     #0B0B12  面板/卡片底色       │
│  Layer 2 — Surface 2     #11111A  悬浮面板/弹窗       │
│  Layer 3 — Surface 3     #181822  激活/悬停状态       │
│  Layer 4 — Surface 4     #20202C  最高层级表面        │
│  Overlay — Glass        rgba(255,255,255,0.03)       │
│  Overlay — Glass Hover  rgba(255,255,255,0.06)       │
│  Overlay — Glass Active rgba(255,255,255,0.10)       │
└─────────────────────────────────────────────────────┘
```

| Token | HEX | 用途 |
|---|---|---|
| `bg-base` | `#07070B` | 页面最底层背景 |
| `bg-surface-1` | `#0B0B12` | 卡片、面板默认背景 |
| `bg-surface-2` | `#11111A` | 悬浮面板、下拉菜单 |
| `bg-surface-3` | `#181822` | 激活态、悬停态背景 |
| `bg-surface-4` | `#20202C` | 最高层级表面 |
| `bg-glass` | `rgba(255,255,255,0.03)` | 玻璃覆盖层 |
| `bg-glass-hover` | `rgba(255,255,255,0.06)` | 玻璃悬停 |
| `bg-glass-active` | `rgba(255,255,255,0.10)` | 玻璃激活 |

### 2.2 品牌色

签名渐变：紫罗兰 → 靛蓝 → 青 — 不是生硬的三色拼接，而是流动的光谱。

      ┃                ┃                ┃
      ┗━━━━━━━━━━━━━━━╋━━━━━━━━━━━━━━━┛
                      ┃
              Signature Gradient
     linear-gradient(135deg,
       #8B5CF6 0%,
       #6366F1 50%,
       #06B6D4 100%)
```

| Token | HEX | 用途 |
|---|---|---|
| `brand-primary` | `#6366F1` | 主品牌色 — 按钮、链接、强调 |
| `brand-secondary` | `#8B5CF6` | 副品牌色 — 渐变起点、辅助强调 |
| `brand-tertiary` | `#06B6D4` | 点缀色 — 渐变终点、高亮提示 |
| `brand-glow` | `rgba(99,102,241,0.35)` | 品牌辉光 — 按钮发光、卡片光晕 |

### 2.7 噪点纹理

极微噪点叠加层，消除渐变条带，增加材质真实感。

  - mix-blend-mode: overlay
  - 覆盖在所有渐变背景之上
```

## 3. 字体系统

### 3.1 字体选择

不使用 SF Pro（Apple）、不使用 Geist（Vercel）、不使用纯 Inter（Linear）— 选择有辨识度但不过于个性的组合。

| 用途 | 字体 | 来源 | 理由 |
|---|---|---|---|
| **Display** | Satoshi | Fontshare (免费) | 几何无衬线、现代、有微妙个性、渲染锐利 |
| **Body / UI** | Inter | Google Fonts | 行业标杆 UI 字体、变量字体、极致可读 |
| **Mono / Code** | JetBrains Mono | Google Fonts | 代码可读性极佳、支持连字、细节优秀 |

### 3.2 字号阶梯

采用 Major Third 比例 (1.2)，但 Hero 区域使用更大的显示尺寸。

| Token | Size | Line-Height | Weight | Tracking | 用途 |
|---|---|---|---|---|---|
| `display-2xl` | 80px | 1.05 | 600 | -0.04em | Hero 主标题 |
| `display-xl` | 64px | 1.08 | 600 | -0.03em | 大型展示标题 |
| `display-lg` | 48px | 1.10 | 600 | -0.02em | 页面标题 |
| `display-md` | 36px | 1.15 | 600 | -0.02em | 区块标题 |
| `heading-lg` | 28px | 1.25 | 600 | -0.01em | 卡片标题 |
| `heading-md` | 22px | 1.30 | 600 | -0.01em | 子标题 |
| `heading-sm` | 18px | 1.35 | 600 | 0 | 小标题 |
| `body-lg` | 18px | 1.65 | 400 | 0 | 大正文（Hero 描述） |
| `body-md` | 16px | 1.60 | 400 | 0 | 默认正文 |
| `body-sm` | 14px | 1.55 | 400 | 0 | 辅助正文 |
| `caption` | 13px | 1.45 | 500 | 0.01em | 标签、时间戳 |
| `code-md` | 14px | 1.65 | 400 | 0 | 代码块 |
| `code-sm` | 13px | 1.60 | 400 | 0 | 行内代码 |

### 3.3 渐变文字

关键标题使用渐变文字效果，而非纯色。

```
Gradient Text:
  background: linear-gradient(135deg, #C4B5FD 0%, #818CF8 50%, #67E8F9 100%);
  background-clip: text;
  -webkit-background-clip: text;
  color: transparent;

  — 使用渐变的亮色端（浅紫→浅蓝→浅青）
  — 确保在深色背景上有足够对比度
  — 仅用于 1-2 个关键词，不整句渐变
```

### 3.4 字重使用

| Weight | 用途 |
|---|---|
| 400 (Regular) | 正文、描述 |
| 500 (Medium) | 标签、按钮、导航 |
| 600 (Semibold) | 标题、强调 |
| 700 (Bold) | 仅用于数字/统计 — 不用于标题 |

## 7. 阴影系统

"Soft Shadow" — 不是生硬的投影，而是多层柔和的阴影 + 内边缘高光。

### 7.1 投影阶梯

| Token | Value | 用途 |
|---|---|---|
| `shadow-xs` | `0 1px 2px rgba(0,0,0,0.08)` | 微小元素 |
| `shadow-sm` | `0 2px 8px rgba(0,0,0,0.12)` | 小卡片 |
| `shadow-md` | `0 4px 16px rgba(0,0,0,0.16)` | 中型卡片 |
| `shadow-lg` | `0 8px 32px rgba(0,0,0,0.24)` | 大卡片、弹窗 |
| `shadow-xl` | `0 16px 48px rgba(0,0,0,0.32)` | 浮空面板 |
| `shadow-2xl` | `0 24px 64px rgba(0,0,0,0.40)` | Hero 卡片 |
| `shadow-glow` | `0 0 40px rgba(99,102,241,0.30)` | 品牌辉光 |
| `shadow-glow-lg` | `0 0 80px rgba(99,102,241,0.20)` | 大范围品牌辉光 |

### 7.2 玻璃复合阴影

玻璃卡片专属阴影 — 外阴影 + 内高光，模拟真实玻璃的边缘光。

```
Glass Card Shadow:
  box-shadow:
    /* 外阴影 — 深度感 */
    0 8px 32px rgba(0,0,0,0.24),
    /* 内高光 — 玻璃边缘光（关键！） */
    inset 0 1px 0 rgba(255,255,255,0.06),
    /* 内底光 — 底部反射光 */
    inset 0 -1px 0 rgba(0,0,0,0.10);

— inset 高光是玻璃质感的核心
— 没有 inset 高光 = 塑料感，不是玻璃
```

### 7.3 悬停阴影变化

```
Default → Hover:
  shadow-md → shadow-lg
  translateY: 0 → -4px
  border: 0.06 → 0.12 opacity
  + shadow-glow (品牌色辉光出现)

— 卡片"浮起来"的物理感
— 不是简单的阴影变大，是位移 + 阴影 + 辉光三重变化
```

## 8. 渐变系统

### 8.1 签名渐变

```
Signature Brand Gradient:
  linear-gradient(135deg, #8B5CF6 0%, #6366F1 50%, #06B6D4 100%)

  Violet ━━━━━━━━━ Indigo ━━━━━━━━━ Cyan

### 8.2 渐变变体

| 名称 | 值 | 用途 |
|---|---|---|

| `gradient-brand-soft` | `linear-gradient(135deg, rgba(139,92,246,0.15), rgba(99,102,241,0.10), rgba(6,182,212,0.05))` | 背景渐变 — 区块底色 |
| `gradient-text` | `linear-gradient(135deg, #C4B5FD, #818CF8, #67E8F9)` | 渐变文字 — 标题关键词 |
| `gradient-border` | `linear-gradient(135deg, rgba(139,92,246,0.3), rgba(6,182,212,0.2))` | 渐变边框 — 强调容器 |
| `gradient-glow-violet` | `radial-gradient(circle, rgba(139,92,246,0.15), transparent 70%)` | 紫色光晕 |
| `gradient-glow-indigo` | `radial-gradient(circle, rgba(99,102,241,0.12), transparent 70%)` | 靛蓝光晕 |
| `gradient-glow-cyan` | `radial-gradient(circle, rgba(6,182,212,0.10), transparent 70%)` | 青色光晕 |

### 8.3 流动渐变

渐变不是静态的，会缓慢流动 — 这是 Nova Studio 的签名动效。

```
Animated Gradient Background:
  - 渐变角度: 135deg
  - 动画: background-position 从 0% 到 200% 再回到 0%
  - 持续时间: 20s
  - 缓动: linear
  - 无限循环

— 极其缓慢，几乎察觉不到，但能"感受到"光在流动
— 营造"活的"背景，不是死板的色块
```

## 9. 玻璃效果系统

"Glass + Blur" — 不是简单的半透明，而是有深度、有质感的玻璃。

### 9.1 玻璃层级

| 层级 | 背景 | 模糊量 | 用途 |
|---|---|---|---|
| `glass-light` | `rgba(255,255,255,0.03)` | `blur(12px)` | 轻量玻璃 — 卡片、面板 |
| `glass-medium` | `rgba(255,255,255,0.05)` | `blur(20px)` | 中量玻璃 — 导航栏、工具栏 |
| `glass-heavy` | `rgba(255,255,255,0.08)` | `blur(32px)` | 重度玻璃 — 弹窗、覆盖层 |
| `glass-nav` | `rgba(7,7,11,0.70)` | `blur(20px)` | 导航专属 — 深色半透明 |

### 9.2 玻璃卡片完整配方

```
Nova Glass Card:
  background: rgba(255,255,255,0.03);
  backdrop-filter: blur(16px) saturate(180%);
  border: 1px solid rgba(255,255,255,0.06);
  border-radius: 24px;
  box-shadow:
    0 8px 32px rgba(0,0,0,0.24),
    inset 0 1px 0 rgba(255,255,255,0.06);

关键细节:
  1. saturate(180%) — 增强玻璃后的颜色饱和度（Apple 技法）
  2. inset 高光 — 玻璃边缘的白色高光线（1px）
  3. 极低背景透明度 (0.03) — 玻璃要"存在"但不"遮挡"
  4. 大圆角 (24px) — 配合玻璃才显柔软
```

### 9.3 玻璃悬停光扫

```
Glass Shimmer on Hover:
  - 一个白色径向渐变从左到右扫过卡片表面
  - opacity: 0 → 0.05 → 0
  - 持续: 800ms
  - 缓动: ease-out
  — 模拟光从玻璃表面划过的效果
```

## 11. 动画系统

### 11.1 缓动曲线

```
  Nova Ease (default):
    cubic-bezier(0.22, 1, 0.36, 1)
    — 快速启动，平滑减速 — Apple 风格

  Nova Spring:
    cubic-bezier(0.34, 1.56, 0.64, 1)
    — 轻微过冲，弹性回弹 — 用于卡片浮起、模态框

  Nova Quick:
    cubic-bezier(0.4, 0, 0.2, 1)
    — 标准缓动 — 用于颜色变化、hover

  Nova Linear:
    linear
    — 仅用于无限循环动画 (渐变流动)
```

### 11.2 时长

| Token | Duration | 用途 |
|---|---|---|
| `instant` | 0ms | 状态切换 |
| `micro` | 100ms | 颜色变化、opacity |
| `fast` | 200ms | 按钮 hover、标签切换 |
| `normal` | 300ms | 卡片 hover、面板展开 |
| `slow` | 500ms | 模态框、抽屉 |
| `slower` | 800ms | Hero 入场、大区域揭示 |
| `ambient` | 20s | 渐变流动（无限循环） |

### 11.3 入场动画

```
Fade In:
  opacity: 0 → 1
  duration: 600ms
  easing: nova-ease

Fade Up:
  opacity: 0 → 1
  translateY: 16px → 0
  duration: 600ms
  easing: nova-ease

Fade Up Large:
  opacity: 0 → 1
  translateY: 24px → 0
  scale: 0.98 → 1
  duration: 800ms
  easing: nova-ease

Scale In:
  opacity: 0 → 1
  scale: 0.96 → 1
  duration: 500ms
  easing: nova-spring
```

### 11.4 交错动画

```
Stagger (用于卡片列表、网格):
  - 每个元素延迟 80ms
  - 第1个: 0ms delay
  - 第2个: 80ms delay
  - 第3个: 160ms delay
  - ...
  - 最大延迟: 500ms (cap)
  - 每个元素: fade-up, 600ms, nova-ease
```

### 11.5 持续动画

| 动画 | 属性 | 持续 | 缓动 | 用途 |
|---|---|---|---|---|
| 渐变流动 | background-position 0%→200%→0% | 20s | linear | Hero 背景 |
| 光晕呼吸 | opacity 0.10→0.15→0.10 | 8s | ease-in-out | 背景光晕 |
| 浮动 | translateY 0→-6px→0 | 3s | ease-in-out | Hero 玻璃卡片 |
| 待办脉冲 | opacity 0.5→1→0.5 | 2s | ease-in-out | "Building" 状态点 |
| 光扫 | translateX -100%→100% | 800ms | ease-out | 玻璃 hover |
| 数字递增 | 0→target | 1200ms | nova-ease | 统计数据 |

### 11.6 滚动揭示

```
Scroll Reveal:
  - 使用 Intersection Observer
  - 元素进入视口 20% 时触发
  - 动画: fade-up, 600ms, nova-ease
  - 一次触发（不重复）
  - 子元素 stagger: 80ms 延迟

  threshold: 0.2
  rootMargin: "0px 0px -100px 0px"
```

  - 内部光晕跟随鼠标位置
  - 鼠标离开: 回正 (nova-spring, 400ms)
```

### 12.3 光标辉光

```
Cursor Glow (暗色区块):
  - 一个 radial-gradient 圆跟随鼠标
  - rgba(99,102,241,0.06), 300px 直径
  - blur(40px)
  - 混合模式: screen
  - 仅在特定区块内生效
  — 营造"光照亮暗处"的氛围
```

### 12.4 命令面板

```
Command Palette (Cmd+K):
  - 从顶部滑入 (translateY(-20px)→0)
  - 玻璃面板 (glass-heavy, blur(32px))
  - radius-2xl, shadow-xl
  - 搜索输入自动聚焦
  - 结果列表: 上下键导航, 回车执行
  - 分类: Actions, Navigate, Search
  - Esc 关闭, 滑出动画
```

### 12.5 平滑滚动

```
Smooth Scroll:
  - CSS: scroll-behavior: smooth
  - 导航锚点: 平滑滚动到对应区块
  - 偏移量: 64px (导航栏高度)
  - duration: 约 800ms (浏览器默认)
```
